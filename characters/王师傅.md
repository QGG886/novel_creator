# 王师傅（厨师）

## 基本信息
- **年龄**：28岁
- **性别**：男
- **职业**：普通厨师 → 膳食官 → 蜀汉民生改革者
- **外貌**：身材敦实，面容老实淳朴，眼神清澈，常穿粗布汉服，给人踏实可靠的感觉

## 性格特征
1. **木讷老实**：不善言辞，不会表达，但做事踏实可靠，让人放心
2. **默默担当**：从不邀功，默默做自己的事，在关键时刻总能挺身而出
3. **善良淳朴**：内心善良，对人真诚，不会算计，容易相信别人
4. **内心细腻**：虽然木讷，但内心细腻，永远在为大家做一些细节上的事，比如给每个人泡好水、整理衣服等
5. **重视友情**：对朋友充满感情，被司马德恩视为唯一的朋友

## 背景故事
现代普通厨师，在一家小餐馆工作，性格木讷老实，不善言辞。穿越前在厨房准备晚餐，是一个普通得不能再普通的人。穿越后，因为木讷老实，一开始被团队忽视，但在司马德恩被抓后，展现出惊人的坚强，成为荆杨（主角）的重要支持力量。作为团队中最普通的人，他的成长和担当让人感动。在三国时期，王师傅凭借精湛的厨艺，成为了蜀汉的膳食官，相当于御厨的地位。王师傅和司马德恩的友情是在穿越初期建立的，穿越初期，大家都被陌生的环境吓坏了，司马德恩和王师傅是唯一保持冷静的人。司马德恩对王师傅说:"别怕，我们一定会找到办法的。"王师傅木讷地点点头，说:"我相信你。"从那时起，司马德恩和王师傅就开始建立了深厚的友情。司马德恩经常和王师傅聊天，虽然大多数时候是司马德恩在说，王师傅在听，但王师傅很享受这种被关注的感觉。司马德恩曾对荆杨说:"王师傅虽然不善言辞，但内心很善良，我第一次感觉到被人平等对待，就是在和他相处的时候。"王师傅听了，内心温暖不已，说:"我也觉得你很好。"王师傅把司马德恩当作自己最信任的朋友，因为这是穿越后第一个真正平等对待他的人。

## 三国时期的"御厨"地位
- **穿越前的平凡**：在现代，王师傅只是一个小餐馆的普通厨师，不算出众。
- **穿越后的地位**：穿越到三国后，王师傅凭借精湛的厨艺，成为了蜀汉的膳食官，相当于御厨的地位。
- **三国时期的饮食水平**：三国时期的饮食水平相对较低，调味料和烹饪技法都很有限。王师傅的现代厨艺在三国时期简直是"降维打击"。
- **对蜀汉的影响**：王师傅的厨艺不仅改善了蜀军和蜀汉高层的伙食，也成为了蜀汉外交和接待的重要手段。东吴和曹魏的使者都对蜀汉的饮食赞不绝口。
- **王师傅的贡献**：王师傅不仅改善伙食，还引入了食品保存技术（如腌制、罐头等），大大提高了蜀汉的食品储备能力。

## 动机
- **内在驱动力**：保护朋友，特别是司马德恩，不辜负诸葛亮的信任
- **外在目标**：在乱世中为蜀汉做出贡献，改善百姓的生活，成为蜀汉的"御厨"

## 能力/技能
- **厨艺精湛**：精通各种菜式，能改善伙食，在三国时期算是"御厨"级别
- **食品保存**：掌握腌制、保存等技术，能制作罐头等现代食品
- **朴实可靠**：做事踏实可靠，让人放心
- **默默付出**：从不邀功，默默做事
- **细节关怀**：内心细腻，永远在为大家做一些细节上的事

## 弱点/缺陷
- **核心弱点**：不善言辞，不会表达，容易被人轻视
- **次要弱点**：过于相信别人，容易被利用
- **性格缺陷**：木讷有时候会错失机会

  - **穿越初期的相遇**:穿越初期，大家都被陌生的环境吓坏了，司马德恩和王师傅是唯一保持冷静的人。司马德恩对王师傅说:"别怕，我们一定会找到办法的。"王师傅木讷地点点头，说:"我相信你。"从那时起，司马德恩和王师傅就开始建立了深厚的友情。

  - **司马德恩看到王师傅的伤口**:司马懿将王师傅带到司马德恩面前，王师傅已经被折磨得遍体鳞伤，但眼神依然清澈。司马德恩看到王师傅的伤口，内心痛苦万分，说:"对不起，连累你了。"王师傅微笑:"不怪你，我们是朋友。"司马德恩流着泪说:"我会保护你的。"王师傅说:"我相信你。"这段对话，成为两人友情最深刻的体现。

  2. 与司马德恩一起被抓捕的痛苦

  4. 司马德恩黑化后继续支持荆杨
  5. 在三国时期成为"御厨"

  2. 第3卷:与司马德恩一起被抓，在司马懿的残酷折磨下展现出惊人的坚强。王师傅虽然被折磨得遍体鳞伤，但始终没有透露任何信息，内心只有一个念头:"我要保护司马德恩，我不会让司马懿利用司马德恩。"这成为两人友情最深刻的体现。

  4. 第5卷:在最终决战中，用自己的方式做出贡献，确保蜀军的粮食供应稳定。最终决战胜利后，蜀汉统一中原，王师傅选择平静的生活，继续在蜀汉做膳食官，改善百姓的饮食。王师傅最终选择回到成都，开了一家小餐馆，过着平静而幸福的生活。

- **最终状态**：一个坚强、可靠、值得信赖的人，虽然依然木讷，但已经成为团队中不可或缺的力量，为蜀汉的民生改善做出巨大贡献。王师傅在三国时期成为了"御厨"，但这对他来说，只是他改善百姓生活的一种方式。王师傅选择回到成都，开了一家小餐馆，过着平静而幸福的生活，经常回忆起穿越后的点点滴滴，特别是和司马德恩的友情。

## 细节关怀的体现
王师傅虽然木讷，但内心细腻，永远在为大家做一些细节上的事：

1. **为每个人泡好水**：每天早上，王师傅都会为每个人泡好水，温度刚好适合每个人。
2. **整理衣服**：王师傅会默默地为每个人整理衣服，确保大家干净整洁。
3. **准备营养餐**：王师傅会根据每个人的身体状况，准备不同的营养餐。比如，为诸葛亮准备清淡的食物，为荆杨准备营养丰富的食物，为孟茹准备她喜欢的甜点。
4. **关心每个人的情绪**：王师傅虽然不善言辞，但会默默观察每个人的情绪。如果有人情绪不好，王师傅会默默为他做一顿他喜欢的菜，用这种方式来安慰他。
5. **记住每个人的喜好**：王师傅会记住每个人的喜好，比如荆杨喜欢吃辣，孟茹喜欢吃甜，李辉辉喜欢吃肉，司马德恩喜欢吃清淡的食物。王师傅会根据每个人的喜好，为他们准备不同的食物。
6. **默默做清洁**：王师傅会默默地为团队做清洁工作，打扫房间，整理物品，让团队的生活环境更加舒适。
