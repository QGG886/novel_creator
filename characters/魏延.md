# 魏延

## 基本信息
- **年龄**：45岁
- **性别**：男
- **职业**：蜀汉大将 → 蜀汉前线指挥官
- **外貌**：身材魁梧，面容刚毅，眼神锐利，常穿厚重铠甲，给人勇猛威武的感觉

## 性格特征
1. **勇猛善战**：武艺高强，勇猛无畏，是蜀汉的核心战将之一
2. **自负骄傲**：对自己充满信心，认为自己的军事才能不输任何人
3. **直率冲动**：说话直来直去，不拐弯抹角，容易得罪人
4. **忠诚不渝**：对刘备和蜀汉忠心不二，愿意为蜀汉战死沙场
5. **矛盾特征**：虽然勇猛善战，但性格急躁，与诸葛亮谨慎的谋略理念常常冲突

## 背景故事
三国时期蜀汉大将，刘备的重要将领。魏延勇猛善战，在刘备时期就立下赫赫战功，镇守汉中多年。刘备死后，魏延继续为蜀汉效力，在北伐中多次立功。但魏延性格自负骄傲，与诸葛亮、杨仪等人的关系紧张。6人团队出现后，魏延从最初的轻视和不屑，到后来的尊重和认可。特别是与司马德恩的矛盾，让魏延的性格冲突更加明显。最终，魏延在与曹魏的战斗中英勇牺牲，用生命证明了自己的忠诚。

## 动机
- **内在驱动力**：证明自己的军事才能，为蜀汉立下更大的战功
- **外在目标**：打败曹魏，完成北伐大业，成为蜀汉第一战将

## 能力/技能
- **武艺高强**：武艺高强，勇猛善战，是一员猛将
- **军事才能**：有一定的军事才能，擅长正面突破和奇袭
- **勇猛无畏**：在战场上勇猛无畏，敢于冲锋陷阵
- **忠诚可靠**：对蜀汉忠心不二，愿意为蜀汉战死沙场

## 弱点/缺陷
- **核心弱点**：自负骄傲，看不起他人，容易得罪人
- **次要弱点**：性格急躁，缺乏耐心，与诸葛亮的谨慎理念冲突
- **性格缺陷**：容易冲动，不计后果

## 关系网络
- **与诸葛亮**：理念冲突，魏延的勇猛与诸葛亮的谨慎常常产生矛盾。诸葛亮虽然欣赏魏延的勇猛，但不太信任魏延的判断。
- **与6人团队**：
  - **与荆杨**：初期轻视荆杨，认为他是个书生不懂军事。后期认可荆杨的谋略能力，开始尊重他。
  - **与孟茹**：孟茹曾开导魏延，魏延虽然嘴硬，但内心感激。
  - **与赵奇**：赵奇曾救治魏延的伤，魏延对赵奇很感激。
  - **与司马德恩**：矛盾重重。司马德恩曾在私下表达对魏延的不满，认为魏延不懂谋略，自己更适合掌权。魏延虽然不知道司马德恩的这番话，但一直对司马德恩心怀不满，认为司马德恩是诸葛亮身边的小人。司马德恩叛逃后，魏延愤怒不已，说："我就知道他不是好人！"
  - **与杨迪**：欣赏杨迪的厨艺，经常夸赞杨迪。
  - **与李辉辉**：李辉辉的活泼让魏延印象深刻，两人关系不错。
- **与刘禅**：忠心辅佐，魏延对刘禅忠心耿耿，愿意为刘禅战死沙场。
- **与姜维**：竞争对手，魏延认为姜维是诸葛亮的心腹，两人关系微妙。

## 角色弧光
- **初始状态**：一个勇猛善战但自负骄傲的蜀汉大将，对6人团队持轻视态度
- **关键挑战**：
  1. 与诸葛亮的理念冲突
  2. 与司马德恩的矛盾
  3. 6人团队出现后的认可问题
  4. 最终的牺牲
- **转变过程**：
  1. 第1卷：对6人团队轻视不屑，认为他们只是一群不知天高地厚的书生。特别是对荆杨，魏延曾私下说："一个书生，懂什么打仗？"
  2. 第2卷：6人团队在蜀汉取得显著成绩，魏延开始改变看法。荆杨的谋略能力让魏延刮目相看，魏延开始尊重荆杨。孟茹曾开导魏延说："魏将军，每个人都有自己的长处。荆杨虽然不是武将，但他的谋略能力不输任何人。"魏延虽然嘴硬，但内心有所触动。
  3. 第3卷：司马德恩在私下表达对魏延的不满，魏延虽然不知道这番话，但一直对司马德恩心怀不满。魏延对荆杨说："那个姓司马的，看着就不像个好人。"荆杨安慰魏延说："司马德恩虽然有时候说话不太好听，但他对蜀汉是有贡献的。"魏延哼了一声："哼，走着瞧。"
  4. 第4卷：司马德恩叛逃后，魏延愤怒不已，说："我就知道他不是好人！当初我就觉得他有问题！"孟茹开导魏延："魏将军，司马德恩的事已经过去了，我们不应该过于纠结。现在最重要的是继续前进，为了蜀汉。"魏延虽然愤怒，但听进了孟茹的话。
  5. 第5卷：在最终决战中，魏延英勇作战，率领蜀军冲破曹魏的防线。但在一次突袭中，魏延被曹魏大军包围，身中数箭，壮烈牺牲。临死前，魏延对身边的士兵说："告诉丞相，魏延没有辜负蜀汉！"魏延用生命证明了自己的忠诚。
- **最终状态**：一个英勇牺牲的蜀汉战将，虽然性格有缺陷，但对蜀汉忠心不二。魏延的死，让蜀汉全军悲痛，诸葛亮也为魏延立碑纪念。魏延虽然在生前与很多人有矛盾，但他最终还是用生命证明了自己对蜀汉的忠诚。

## 标志性特征
- **台词风格**：说话直率冲动，声音洪亮，经常说"末将这就去办"、"这点小事算什么"、"老子去杀了他"
- **习惯动作**：思考时会下意识地摩挲剑柄，或者在战场上握紧武器
- **口头禅**："老子去杀了他"（面对敌人时经常说），"这点小事算什么"（不屑时经常说）
- **与司马德恩的矛盾**：魏延一直对司马德恩心怀不满，司马德恩叛逃后，魏延愤怒不已，证明了自己的判断
- **与荆杨的关系**：从轻视到尊重，魏延认可荆杨的谋略能力，两人成为战友
