# 赵奇（医生）

## 基本信息
- **年龄**：35岁
- **性别**：男
- **职业**：现代医生 → 蜀汉御医 → 医疗体系建立者
- **外貌**：身材修长儒雅，面容温和，眼神沉稳，常穿素净汉服，给人可靠和专业的感觉

## 性格特征
1. **儒雅沉稳**：永远温文尔雅，说话慢条斯理，不急不躁，给人安全感
2. **文绉绉的说话方式**：讲话带点文绉绉，不是呆板，是真·老派知识分子气质，让人觉得有文化底蕴
3. **医者仁心**：对病人充满同情和关爱，不分敌我，救治所有需要帮助的人
4. **理性冷静**：面对紧急情况能保持冷静，理性分析，是团队的"定海神针"
5. **温和包容**：对李辉辉的调侃总是包容回应，像长辈一样照顾团队成员

## 背景故事
现代医生，穿越前在医院工作，是一名优秀的外科医生。穿越后，通过现代医疗知识救治诸葛亮，成为蜀汉的御医。作为团队中最年长者之一，他像父亲/长者一样照顾团队中的每一个人。李辉辉背地里叫他"老头"，但他从不生气，反而温和回应。他用自己的医术和温和的性格，成为团队中不可或缺的稳定力量。

## 与李辉辉的互动
- **李辉辉的调侃**:李辉辉经常在背地里叫赵奇"老头"，偶尔当面也开玩笑地说"老头，你这个方法太老派了"。
- **赵奇的包容**:赵奇从不生气，总是温和地回应:"辉辉说得是，老派方法虽慢，但稳重。"
- **两人的关系**:李辉辉虽然嘴上调侃，但内心非常敬重赵奇，经常在赵奇面前像学生一样请教问题。赵奇也把李辉辉当后辈照顾，两人的关系亲密和谐。

## 动机
- **内在驱动力**：救治更多病人，建立完善的医疗体系，保护团队成员
- **外在目标**：在三国乱世中推广现代医疗理念，建立三国版医疗体系

## 能力/技能
- **现代医学**：精通现代外科手术和药物治疗
- **青霉素提炼**：能提炼青霉素等现代药物
- **临床经验**：丰富的临床经验，能应对各种医疗情况
- **教学能力**：善于传授医术，培养新的医生
- **冷静分析**：面对紧急情况能理性分析，冷静处理

## 弱点/缺陷
- **核心弱点**：年龄较大（相对于其他团队成员），体力不如年轻人
- **次要弱点**：过于温和，有时候不够果断
- **性格缺陷**：过于理性，有时候会忽略情感因素

## 角色弧光
- **初始状态**：一个儒雅沉稳的现代医生，说话文绉绉，医术高超
- **关键挑战**：
  1. 在三国乱世中施展医术，建立医疗体系
  2. 救治诸葛亮的巨大压力
  3. 救治刘禅获得信任
  4. 团队成员受伤时的紧急救治
  5. 司马德恩背叛后团队的情感支持
- **转变过程**：
  1. 第1卷:救治诸葛亮和蜀军伤病员，建立初步信任。曾救治生病的刘禅，获得刘禅感激和信任。与诸葛亮成为知己，经常下棋深谈。

  2. 第2卷:开始建立蜀汉医疗体系，培养医疗人才。与诸葛亮讨论医兵相通的道理，两人相谈甚欢。

  3. 第3卷:司马德恩和杨迪被抓，赵奇虽然担心，但保持冷静，支持团队继续前进。对李辉辉说:"辉辉，我们不能因为两人被抓就乱了阵脚，要有计划地营救。"

  4. 第4卷:司马德恩背叛后，像父亲一样照顾团队，特别是杨迪。赵奇对杨迪说:"德恩的事，不是你的错。你坚强起来，德恩如果看到，也会欣慰。"

  5. 第5卷:在最终决战中，负责医疗救援，救治伤员。最终决战胜利后，继续在蜀汉建设医疗体系，培养更多医疗人才。

- **最终状态**：建立蜀汉完整医疗体系，培养大批医疗人才，在新历史中继续贡献。
