# 张医生（外科医生）

## 基本信息
- **年龄**：35岁
- **性别**：男
- **职业**：刚当上外科主治医师 → 御医 → 蜀汉医疗改革者
- **外貌**：身材中等，头发乌黑，面容成熟稳重，眼神温和，常穿白色汉服，给人可靠和值得信赖的感觉

## 性格特征
1. **医者仁心**：对病人充满同情和关爱，不分敌我，救治所有需要帮助的人
2. **沉稳可靠**：面对紧急情况能保持冷静，给人安全感，是团队的"定海神针"
3. **博学多才**：不仅精通现代医学，还精通家传中医，家学渊源，人生经验丰富
4. **刚当上主治医师的成就感**：刚当上外科主治医师，对未来充满期待，这成为了他穿越后努力的动力
5. **矛盾特征**：虽然医术高超，但对现代医疗手段与古代理念的冲突有时会感到矛盾

## 背景故事
现代外科主治医师，刚当上外科主治医师不久，对事业充满热情和期待。家传中医，祖上曾与华佗有交往，家传医术或许有华佗的传承。穿越前在医院工作，刚被任命为外科主治医师，这是他职业生涯的重要里程碑。穿越后，利用现代医疗知识和家传中医救治诸葛亮和蜀军伤病员，成为团队中的核心智囊之一。作为团队中最年长者，他像父亲一样照顾团队中的每一个人。刚当上主治医师的成就感，让张医生在三国时期更加努力，希望能够实现自己的医疗理想。后期揭示家传中医的秘密，为蜀汉的医疗改革做出巨大贡献。

## 刚当上主治医师的成就感
1. **职业生涯的里程碑**：
   - 张医生刚当上外科主治医师，这是他职业生涯的重要里程碑。
   - 从实习医生到住院医师，再到主治医师，张医生经过了多年的努力。
   - 刚当上主治医师，张医生对未来充满期待，希望能够实现自己的医疗理想。

2. **对事业的热情**：
   - 张医生对外科手术充满热情，每一次手术都认真对待。
   - 刚当上主治医师，张医生希望能够有更多的机会施展自己的才华。
   - 穿越，对张医生来说，既是一个挑战，也是一个实现医疗理想的机会。

3. **穿越前的兴奋**：
   - 穿越前，张医生正在准备一次重要的手术，这是他当上主治医师后的第一次重要手术。
   - 张医生对这次手术充满期待，希望能够证明自己的能力。
   - 穿越后，张医生虽然失去了这次手术的机会，但在三国时期，他有了更多的机会施展自己的才华。

4. **穿越后的动力**：
   - 刚当上主治医师的成就感，成为了张医生在三国时期努力的动力。
   - 张医生说："我刚当上主治医师，本来可以大展身手，但穿越到了三国。不过，在这里，我也许能够实现更大的医疗理想。"
   - 张医生在三国时期救治了更多的人，实现了自己的医疗理想。

## 华佗传承详解(重要伏笔)
- **传承渊源**:张医生的祖上是华佗的徒弟之一，在华佗被曹操处死前，华佗将部分医书和医术传授给了徒弟，让他们传承下去。这些医书包括《青囊书》的残卷和华佗的一些独门医术，如五禽戏、麻沸散的配方、外科手术技法等。华佗对徒弟说:"这些医术，你一定要传承下去，不要让它们失传。记住，医术是用来救人的，不是用来求名的。"徒弟郑重地答应了华佗。

- **家传中医**:张医生的祖上将这些医书和医术代代相传，成为家传中医。每一代的传人都必须学习这些医术，并选择一个继承人继续传承。张医生从小学习家传中医，虽然他没有全部掌握，但对一些关键医术有所了解，特别是外科手术技法和一些草药的使用方法。

- **现代医学与家传中医的结合**:张医生在现代学习的是西医外科，但家传中医也给了他一些启发，特别是在草药使用和手术技法方面。张医生曾经尝试将西医和中医结合起来，取得了一些成果，但因为现代医疗体系的不同，这些尝试没有得到重视。

- **穿越后的应用**:穿越后，张医生利用现代医疗知识和家传中医救治诸葛亮和蜀军伤病员。在现代医疗条件受限的情况下，家传中医发挥了重要作用。张医生用家传医术提炼青霉素，这是现代医疗知识和家传中医结合的典范。

- **秘密的揭示**:第3卷，张医生在救治一位重症病人时，用家传医术取得奇效，引起了诸葛亮的注意。诸葛亮询问:"你的医术从何而来?似乎不似普通中医。"张医生沉默片刻，决定说出真相:"不瞒丞相，我的家传中医，或许与华佗先生有关。"诸葛亮大惊:"与华佗先生有关?"张医生点头:"是的，我的祖上是华佗先生的徒弟，华佗先生在去世前，将部分医书和医术传授给了我的祖上。"诸葛亮激动不已:"这么说，你拥有华佗先生的部分医术传承?"张医生谦虚地说:"只能说略知一二，并没有全部掌握。"诸葛亮感慨:"华佗先生的医术，一直是我心中的遗憾。想不到，竟然在你这里找到了传承。"

- **医书的内容**:张医生的祖上传承下来的医书包括《青囊书》的残卷、华佗的一些独门医术和一些草药配方。这些医书虽然不是完整的《青囊书》，但已经足够让张医生在三国乱世中发挥重要作用。张医生将这些医书的内容记在脑海中，虽然没有带在身边，但已经足够使用。

- **华佗的精神**:张医生一直牢记华佗的教诲:"医术是用来救人的，不是用来求名的。"这句话也成为张医生的座右铭，指导他的行医理念。张医生在救治诸葛亮和蜀军伤病员时，从不求名利，只是希望能够救人。这种精神，让诸葛亮和其他人都对张医生充满敬意。

- **对蜀汉的贡献**:张医生揭示家传中医的秘密后，开始为蜀汉培训医疗人才，推广中西医结合的医疗理念，改革蜀汉的医疗体系。张医生利用家传医书的内容，编写了新的医学教材，为蜀汉培养了大批医疗人才，大大提高了蜀汉的医疗水平。张医生的贡献，为蜀汉的国力增强做出了重要贡献。

- **与诸葛亮的交流**:张医生和诸葛亮多次交流医术和治国理念，两人相谈甚欢。诸葛亮对张医生说:"你的家传医术，不仅救了无数人的命，也为蜀汉的未来打下了基础。我代表蜀汉人民，感谢你。"张医生谦虚地说:"这是我应该做的，医者父母心，救人是我的责任。"诸葛亮说:"你是一个真正的医者，华佗先生若在天有灵，一定会为你感到骄傲。"

- **传承的未来**:张医生决定将家传医术传授给更多的人，让华佗的医术在三国时期发扬光大。张医生说:"华佗先生的医术，不应该只属于我，应该属于天下人。我会将这些医术传授给更多的人，让它们造福更多的人。"诸葛亮说:"这是一个伟大的决定，蜀汉会全力支持你。"

## 动机
- **内在驱动力**：医者仁心，救治更多病人，传承家传医术，实现自己的医疗理想
- **外在目标**：在三国乱世中拯救更多生命，推广中西医结合的医疗理念

- **现代医学**：精通现代外科手术和药物治疗
- **家传中医**：精通家传中医，或许有华佗的传承
- **青霉素提炼**：能提炼青霉素等现代药物
- **临床经验**：丰富的临床经验，能应对各种医疗情况
- **教学能力**：善于传授医术，培养新的医生

## 弱点/缺陷

- **次要弱点**：对团队过于保护，有时会过度担心
- **性格缺陷**：过于理想主义，有时会忽视现实的残酷
