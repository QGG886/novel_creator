# 模板系统

## 使用说明
本文件提供小说创作中常用模板，包括大纲模板、角色模板、章节模板等，帮助快速开始创作。

## 章节模板

### 章节结构模板
```markdown
# 第X章 章标题

## 开头（100-300字）
[如果是第一章：介绍场景/主角/引发事件]
[如果不是第一章：承接上文，说明时间地点]

## 发展（篇幅最大）
- 引入冲突或任务
- 角色展开行动
- 小挫折或小胜利
- 推进剧情

## 高潮（章节关键点）
- 最大冲突或转折
- 角色面临抉择
- 揭露重要信息

## 结尾（100-200字）
- 确认本章成果
- 为下章留线索/悬念
```

## 检查清单模板

### 章节检查清单
```markdown
# 章节检查清单

## 内容检查
- [ ] 章节目标达成
- [ ] 情节推进合理
- [ ] 冲突有意义
- [ ] 对话自然流畅

## 文风检查
- [ ] 遵循文风指南
- [ ] 语言表达准确
- [ ] 句式符合规范

## 连贯性检查
- [ ] 与前后章节衔接
- [ ] 角色行为一致
- [ ] 时间地点准确
- [ ] 无逻辑矛盾

## 字数检查
- [ ] 字数符合要求
- [ ] 长度适当
```

### 完成检查清单
```markdown
# 完成检查清单

## 内容完整性
- [ ] 大纲完整
- [ ] 角色完整
- [ ] 世界观完整（如需要）
- [ ] 所有章节完成

## 质量检查
- [ ] 情节连贯
- [ ] 角色一致
- [ ] 设定一致
- [ ] 主题清晰

## 格式检查
- [ ] 章节编号正确
- [ ] 标题格式统一
- [ ] 文件命名规范

## 导出检查
- [ ] 创建导出版本
- [ ] 验证导出内容
- [ ] 备份完成版本
```

## 使用建议
1. 根据项目需求选择合适的模板
2. 模板可以根据需要修改和定制
3. 定期检查清单，确保创作质量
4. 使用进度报告模板跟踪进度
5. 保留版本信息，便于版本管理
