# 节奏分析检查Skill

## 功能

检查章节的叙述节奏是否合理，发现问题并提供具体的修改方案。

---

## 检查维度

### 1. 信息密度
- 信息是否过于密集，让读者疲劳
- 信息是否过于稀疏，缺乏内容
- 信息分布是否均匀

### 2. 动作密度
- 动作场景的分布是否合理
- 动作描写是否充分
- 静态和动态是否平衡

### 3. 情感强度
- 情感强度变化是否合理
- 情感积累是否充分
- 情感爆发是否合理

### 4. 冲突强度
- 冲突的出现和解决是否合理
- 冲突强度是否递进
- 冲突类型是否多样

### 5. 高潮位置
- 高潮位置是否合适（理想位置75%-85%）
- 高潮前的铺垫是否充分
- 高潮后的回落是否自然

## 修改方案生成规则

### 节奏过慢（信息密度低）

**问题示例**：大段环境描写和心理活动，情节推进缓慢

**修改方案**：
- 方案类型：压缩
- 具体操作：删除冗余，合并内容
- 修改后内容：
  - 原文（200字）：详细描写环境
  - 修改后（80字）：保留关键信息，删除细节
- 操作说明：
  1. 删除重复的描述
  2. 合并相似内容
  3. 使用短句加快节奏
  4. 删除不必要的形容词

### 节奏过快（信息密度高）

**问题示例**：情节推进太快，缺乏细节，读者来不及消化

**修改方案**：
- 方案类型：扩展
- 具体操作：增加细节描写
- 修改后内容：
  - 原文（50字）：快速叙述事件
  - 修改后（150字）：增加细节和描写
- 操作说明：
  1. 增加环境细节
  2. 扩展心理活动
  3. 增加动作细节
  4. 使用长句放慢节奏

### 高潮位置不合理

**问题示例**：高潮出现在50%，太早

**修改方案**：
- 方案类型：调整
- 具体操作：调整情节顺序
- 修改后内容：
  - 将高潮情节移到75%-85%位置
  - 在高潮前增加铺垫内容
- 操作说明：
  1. 识别高潮情节
  2. 计算当前位置百分比
  3. 调整内容顺序或增加铺垫

### 情感强度不足

**问题示例**：重要情感场景描写平淡

**修改方案**：
- 方案类型：增加
- 具体操作：增强情感表达
- 修改后内容：
  - 原文："他很难过。"
  - 修改后："他心如刀绞，泪水夺眶而出。"
- 操作说明：
  1. 增加心理描写
  2. 增加生理反应描写
  3. 增加环境烘托
  4. 使用比喻、排比等修辞

## 检查清单

- [ ] 整体节奏适合章节类型
- [ ] 信息密度适中
- [ ] 动作和静态平衡
- [ ] 情感强度递进合理
- [ ] 高潮位置在75%-85%
- [ ] 高潮前铺垫充分
- [ ] 节奏有变化，不单调

---

## 严重程度分级

### A类（严重）
- 高潮位置严重偏离（<50%或>95%）
- 整章节奏完全不适合章节类型
- 重大情节节点节奏处理不当

### B类（中等）
- 局部节奏过快或过慢
- 信息密度不均匀
- 情感强度递进不够流畅

### C类（轻微）
- 个别段落可以更紧凑
- 个别描写可以精简

---

## 注意事项

1. **章节类型匹配**：动作场景节奏快，情感场景节奏慢
2. **保持连贯**：调整节奏时要保持情节连贯
3. **高潮位置**：理想位置75%-85%，但可以灵活调整
4. **渐进变化**：节奏变化要渐进，不要突兀
5. **具体可操作**：修改方案必须具体，能直接执行
