# 对话自然化工具

## 功能
优化对话的自然度和真实性，使对话更符合角色性格和情境。

## 输入格式
```json
{
  "原始对话": "包含对话的段落",
  "场景信息": {
    "地点": "",
    "时间": "",
    "氛围": "",
    "正式程度": "正式/半正式/随意"
  },
  "参与角色": {
    "角色A": {
      "性格": "",
      "身份": "",
      "教育水平": "高/中/低",
      "说话习惯": ""
    }
  }
}
```

## 输出格式
```json
{
  "状态": "成功",
  "优化后对话": {
    "完整文本": "",
    "纯对话": ""
  },
  "改动说明": {
    "主要改动": ["改动1", "改动2"]
  }
}
```

## 优化维度
1. 口语化表达
2. 句式变化
3. 语气和停顿
4. 角色个性
5. 情境匹配
6. 互动自然

## 注意事项
1. 保持可读性和真实性的平衡
2. 区分角色之间的语言差异
3. 注意对话的连贯性
4. 不要让角色说作者想说的话
