# 主题分析检查Skill

## 功能

检查章节是否体现主题，主题表达是否一致、深刻，发现问题并提供具体的修改方案。

---

## 检查维度

### 1. 核心主题
- 是否体现核心主题
- 主题表达是否明确

### 2. 次要主题
- 是否体现次要主题（如有）
- 次要主题是否与核心主题协调

### 3. 隐含主题
- 是否有隐含主题（如有）
- 隐含主题是否巧妙

### 4. 主题深度
- 主题表达是否有深度
- 是否流于表面

### 5. 主题一致性
- 主题表达是否一致
- 是否有矛盾或跑题

### 6. 主题层次
- 主题层次是否清晰
- 表达是否递进

---

  }

## 修改方案生成规则

### 主题缺失

**问题示例**：章节未体现设定的核心主题

**修改方案**：
- 方案类型：增加
- 具体操作：增加体现主题的内容
- 修改后内容：
  - 在关键情节中增加主题相关的内容
  - 通过角色对话、心理活动体现主题
  - 在结尾处点题
- 操作说明：
  1. 识别章节中的关键情节
  2. 在这些情节中增加主题元素
  3. 通过角色言行体现主题
  4. 在结尾处点明主题

### 主题浅显

**问题示例**：主题表达流于表面，缺乏深度

**修改方案**：
- 方案类型：深化
- 具体操作：深化主题表达
- 修改后内容：
  - 原文："他明白了友情的可贵。"
  - 修改后："他终于明白，友情不是锦上添花，而是雪中送炭。真正的朋友，是在你最困难时依然站在你身边。"
- 操作说明：
  1. 增加具体事例支撑主题
  2. 通过对比、隐喻深化主题
  3. 从多个角度阐述主题
  4. 避免说教，用情节体现

### 主题矛盾

**问题示例**：章节中体现的主题与全书主题矛盾

**修改方案**：
- 方案类型：调整
- 具体操作：调整与主题矛盾的内容
- 修改后内容：
  - 删除或修改矛盾的内容
  - 调整情节走向
  - 修改角色言行
- 操作说明：
  1. 识别矛盾的具体内容
  2. 分析为什么矛盾
  3. 调整使其符合主题

## 严重程度分级

### A类（严重）
- 完全偏离核心主题
- 主题表达与全书矛盾
- 主题完全缺失

### B类（中等）
- 主题表达不够明确
- 主题深度不够
- 主题层次不清晰

### C类（轻微）
- 主题表达可以更深化
- 个别内容与主题关联不强
