# 情感曲线分析Skill

## 功能
分析故事和角色的情感发展轨迹，绘制情感曲线，评估情感体验的效果。

## 使用场景
- 大纲设计时，规划情感发展
- 章节创作时，检查情感表达
- 角色塑造时，优化情感变化
- 读者反馈情感问题时，进行分析

## 情感维度

### 1. 整体情感
- 主导情感（欢乐、悲伤、愤怒等）
- 情感基调（轻松、严肃、温暖、黑暗等）
- 情感氛围
- 情感主题

### 2. 角色情感
- 主角情感变化
- 配角情感发展
- 角色间情感互动
- 情感冲突

### 3. 读者情感
- 期待感
- 紧张感
- 同情心
- 满足感
- 情感共鸣

### 4. 情感变化
- 情感转换
- 情感起伏
- 情感积累
- 情感爆发
- 情感沉淀

## 情感类型

### 正面情感
- 喜悦
- 温馨
- 希望
- 感动
- 激动
- 满足
- 宁静

### 负面情感
- 悲伤
- 愤怒
- 恐惧
- 焦虑
- 失望
- 绝望
- 压抑

### 复杂情感
- 矛盾
- 悖离
- 惆怅
- 迷茫
- 不安
- 忧郁

## 情感弧光类型

### 成长弧光
- 初始状态：天真、无邪、简单
- 遭遇挑战：困惑、挣扎、痛苦
- 逐渐成长：理解、成熟、深刻
- 最终状态：智慧、坚韧、完整

### 救赎弧光
- 初始状态：堕落、错误、迷失
- 觉醒时刻：愧疚、悔恨、觉醒
- 寻求救赎：努力、改变、付出
- 最终状态：救赎、和解、升华

### 堕落弧光
- 初始状态：善良、正义、理想
- 遭遇诱惑/打击：动摇、迷失、妥协
- 逐渐堕落：邪恶、残忍、扭曲
- 最终状态：堕落、毁灭、绝望

### 恢复弧光
- 初始状态：完整、正常
- 遭遇创伤：受伤、破碎、迷茫
- 治愈过程：挣扎、接受、成长
- 最终状态：恢复、升华、新生

## 情感技巧

### 1. 情感铺垫
- 逐步积累情感
- 细节暗示情感
- 氛围营造情感
- 对话透露情感

### 2. 情感爆发
- 情感积累到顶点
- 选择恰当的爆发时机
- 通过冲突激发情感
- 使用强烈表达方式

### 3. 情感转换
- 自然的情感转换
- 合理的情感变化
- 逐渐的情感调整
- 突然的情感转折

### 4. 情感共鸣
- 创造可共情的角色
- 使用普遍的情感体验
- 细节引发共鸣
- 情境唤起共鸣

### 5. 情感对比
- 情感的强烈对比
- 不同角色的情感对比
- 情感的对比变化
- 对比增强情感效果

## 工作流程

### 第一步：情感识别
1. 阅读内容识别情感
2. 标记情感类型
3. 评估情感强度
4. 记录情感位置

### 第二步：情感追踪
1. 追踪整体情感变化
2. 追踪角色情感发展
3. 追踪情感冲突
4. 追踪情感高潮

### 第三步：曲线绘制
1. 绘制整体情感曲线
2. 绘制角色情感弧光
3. 标记关键情感节点
4. 分析情感变化规律

### 第四步：效果评估
1. 评估情感表达效果
2. 评估情感连贯性
3. 评估情感感染力
4. 评估情感共鸣度

### 第五步：问题分析
1. 识别情感问题
2. 分析问题原因
3. 评估问题影响
4. 提供解决方案

## 注意事项
- 情感要真实，不过度煽情
- 情感要合理，有充分依据
- 情感要丰富，有层次变化
- 情感要自然，不突兀
- 情感要连贯，前后呼应
- 情感要适度，不过度渲染
- 考虑读者的情感承受能力
- 注意情感与内容的匹配
