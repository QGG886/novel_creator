# 文风格式化工具

## 功能
将文风信息格式化为结构化的、可指导写作的文风指南。

## 使用场景
- Agent分析完范文后，生成文风指南
- Agent收集完用户文风偏好后，格式化成指南
- 需要将文风信息保存为标准格式时

## 输入格式
```json
{
  "文风信息": {
    "基础信息": {
      "文风名称": "如：古龙风、村上春树风",
      "主要特征": "一句话概括",
      "适用类型": "言情、玄幻等"
    },
    "语言特征": {
      "用词风格": "华丽/朴实/典雅/口语化",
      "句式特点": ["句式示例1", "句式示例2"],
      "常用修辞": ["比喻", "拟人", "夸张"]
    },
    "叙述特征": {
      "叙事视角": "第一人称/第三人称",
      "叙述节奏": "紧凑/舒缓/快慢结合",
      "心理描写": "深入/浅层/间接"
    },
    "情感特征": {
      "基调": "幽默/严肃/悲伤/温馨",
      "表达方式": "直白/含蓄/克制"
    },
    "结构特征": {
      "段落组织": "长段落/短段落/混合",
      "对话比例": "高/中/低",
      "描写叙述比": "x:x"
    },
    "示例段落": [
      {
        "类型": "描写段落",
        "示例": "摘自范文或生成的示例",
        "特点说明": "为什么是典型"
      }
    ]
  }
}
```

## 输出格式
```json
{
  "状态": "成功",
  "文风指南": {
    "基础信息": {...},
    "语言特征": {...},
    "叙述特征": {...},
    "情感特征": {...},
    "结构特征": {...},
    "示例段落": [...]
  },
  "文件保存": {
    "路径": "output/style_guide.json",
    "格式": "JSON"
  }
}
```

## 注意事项
1. 本工具只负责格式化，不负责文风分析
2. 所有文风信息必须由Agent提供
3. 示例段落必须真实，不能杜撰
4. 输出保存路径：`output/style_guide.json`

## 文风指南模板

### Markdown格式模板
```markdown
# 文风指南

## 语言特征
- **用词风格**：华丽/朴实/典雅/口语化
- **句式特点**：[句式描述]
- **常用修辞**：[修辞列表]

## 叙述特征
- **叙事视角**：第一人称/第三人称
- **叙述节奏**：紧凑/舒缓/快慢结合
- **心理描写**：深入/浅层/间接

## 情感特征
- **基调**：幽默/严肃/悲伤/温馨
- **表达方式**：直白/含蓄/克制

## 结构特征
- **段落组织**：长段落/短段落/混合
- **对话比例**：高/中/低
- **描写叙述比**：x:x

## 写作规范
- **禁用表达**：[列表]
- **推荐表达**：[列表]

## 示例段落
### 描写段落
[示例文本]
**特点说明**：[为什么是典型]

### 对话段落
[示例文本]
**特点说明**：[为什么是典型]

### 动作段落
[示例文本]
**特点说明**：[为什么是典型]
```
