# 版本管理工具

## 功能
管理小说创作的多个版本，支持版本记录、对比、恢复和导出。

## 输入格式
```json
{
  "任务类型": "创建版本/对比版本/恢复版本/查看历史",
  "版本信息": {
    "版本名称": "v1.0",
    "版本说明": ""
  },
  "当前内容": {
    "大纲": "",
    "章节": []
  }
}
```

## 输出格式
```json
{
  "状态": "成功",
  "版本ID": "唯一标识",
  "版本信息": {
    "版本名称": "",
    "创建时间": "",
    "版本说明": ""
  }
}
```

## 版本类型
1. 里程碑版本
2. 编辑版本
3. 实验版本

## 注意事项
1. 定期创建版本备份
2. 为每个版本添加清晰说明
3. 版本恢复前创建备份
