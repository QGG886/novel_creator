# 主题分析Skill

## 功能
分析故事的主题表达，识别核心主题，评估主题的深度和表达效果。

## 使用场景
- 大纲设计时，确定主题方向
- 章节创作时，检查主题表达
- 完成部分内容后，评估主题
- 读者反馈主题问题时，进行分析

## 主题类型

### 1. 核心主题
- 故事最根本要表达的主题
- 贯穿全文的主题思想
- 主旨和中心思想
- 核心价值观

### 2. 次要主题
- 辅助核心主题的主题
- 分主题或子主题
- 从不同角度探讨的主题
- 支撑核心主题的主题

### 3. 隐含主题
- 未直接表达的深层主题
- 通过情节和角色暗示的主题
- 需要读者自己体会的主题
- 含蓄表达的主题

## 主题分析维度

### 1. 主题清晰度
- 主题是否明确
- 主题是否容易理解
- 主题表达是否清晰
- 主题是否贯穿全文

### 2. 主题深度
- 主题的思想深度
- 主题的层次感
- 主题的普适性
- 主题的独特性

### 3. 主题表达
- 表达方式是否多样
- 表达是否有力
- 表达是否自然
- 表达是否完整

### 4. 主题一致性
- 前后是否一致
- 与内容是否匹配
- 主题是否完整
- 是否有矛盾

### 5. 主题影响力
- 情感冲击力
- 思想启发性
- 读者共鸣度
- 社会价值

## 主题表达方式

### 1. 通过情节表达
- 核心冲突体现主题
- 情节发展展现主题
- 结局表达主题
- 关键事件强化主题

### 2. 通过角色表达
- 角色弧光体现主题
- 角色选择反映主题
- 角色关系表达主题
- 角色成长展现主题

### 3. 通过对话表达
- 关键对话阐述主题
- 辩论探讨主题
- 内心独白表达主题
- 象征对话暗示主题

### 4. 通过描写表达
- 场景描写烘托主题
- 细节描写暗示主题
- 氛围营造强化主题
- 象征手法表达主题

### 5. 通过对比表达
- 角色对比体现主题
- 事件对比展现主题
- 场景对比强化主题
- 时间对比深化主题

## 工作流程

### 第一步：主题识别
1. 阅读分析内容
2. 识别核心主题
3. 发现次要主题
4. 探索隐含主题
5. 分析主题层次

### 第二步：主题分析
1. 分析主题的清晰度
2. 评估主题的深度
3. 检查主题的表达
4. 验证主题的一致性
5. 评估主题的影响力

### 第三步：主题追踪
1. 追踪主题的分布
2. 记录主题的表现
3. 标记主题的强化
4. 识别主题的缺失
5. 评估主题的发展

### 第四步：问题分析
1. 识别主题问题
2. 分析问题原因
3. 评估问题影响
4. 提供解决方案

## 注意事项
- 主题要自然流露，不要说教
- 主题要与故事内容匹配
- 主题不要过于复杂或抽象
- 主题要贯穿全文，不要半途而废
- 可以有多个主题，但要有一个核心主题
- 主题表达要多样，不要单一
- 主题可以通过多种方式表达
- 主题要有深度，但不要故弄玄虚
