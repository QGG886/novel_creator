# 世界观一致性检查工具

## 功能
检查世界观设定在全文中的一致性，识别设定矛盾。

## 输入格式
```json
{
  "世界观文档": "完整的世界观文档",
  "全部章节": ["章节1", "章节2"],
  "检查维度": ["地理", "历史", "社会", "技术"]
}
```

## 输出格式
```json
{
  "状态": "成功",
  "问题清单": [
    {
      "设定项": "",
      "位置对比": {
        "位置1": "",
        "位置2": ""
      },
      "矛盾描述": ""
    }
  ]
}
```

## 检查维度
1. 地理环境
2. 历史背景
3. 社会结构
4. 文化传统
5. 技术/魔法系统
6. 种族/派系
