# 多格式导出工具

## 功能
将小说内容导出为多种格式。

## 支持格式
- Markdown (.md)
- 纯文本 (.txt)
- HTML (.html)
- PDF (.pdf)
- EPUB (.epub)
- DOCX (.docx)

## 输入格式
```json
{
  "导出格式": "Markdown/文本/HTML/PDF/EPUB/DOCX",
  "导出范围": "全部/指定章节",
  "包含内容": {
    "大纲": true,
    "角色": true,
    "章节": true
  }
}
```

## 输出格式
```json
{
  "状态": "成功",
  "导出格式": "",
  "输出文件": "文件路径",
  "文件大小": "大小"
}
```

## 注意事项
1. 确保导出前内容完整
2. 检查导出后的格式
3. 验证文件完整性
