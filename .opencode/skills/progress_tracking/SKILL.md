# 进度追踪工具

## 功能
追踪小说创作的整体进度，记录完成情况。

## 输入格式
```json
{
  "任务类型": "进度报告/更新进度/规划进度",
  "目标设定": {
    "总章节数": 30,
    "总字数目标": 100000
  },
  "当前状态": {
    "已完成章节": ["第1章"],
    "当前字数": 15000
  }
}
```

## 输出格式
```json
{
  "状态": "成功",
  "进度报告": {
    "章节进度": {
      "已完成": 10,
      "完成百分比": "33.3%"
    },
    "字数进度": {
      "当前字数": 15000,
      "完成百分比": "15%"
    }
  }
}
```

## 进度维度
1. 章节进度
2. 字数进度
3. 里程碑进度
4. 时间进度

## 注意事项
1. 定期更新进度记录
2. 保持进度记录的准确性
3. 根据实际情况调整目标
