# 情感分析检查Skill

## 功能

检查章节的情感表达是否充分、合理、自然，发现问题并提供具体的修改方案。

---

## 检查维度

### 1. 整体情感
- 主导情感是否清晰
- 情感基调是否一致

### 2. 角色情感
- 角色情感变化是否合理
- 情感是否体现角色性格

### 3. 情感转换
- 情感转换是否自然
- 转换是否有铺垫

### 4. 情感积累
- 情感积累是否充分
- 积累过程是否合理

### 5. 情感爆发
- 情感爆发是否合理
- 爆发时机是否合适

### 6. 情感共鸣
- 是否能引起读者情感共鸣

## 修改方案生成规则

### 情感转换生硬

**问题示例**：角色突然从平静变为愤怒，无铺垫

**修改方案**：
- 方案类型：增加
- 具体操作：增加情感转换的过渡
- 修改后内容：
  - 原文："他突然大怒。"
  - 修改后："他先是皱眉，继而握紧拳头，终于忍不住大吼起来。"
- 操作说明：
  1. 增加中间情感状态（皱眉→握拳→大吼）
  2. 添加触发事件
  3. 增加心理描写

### 情感表达不足

**问题示例**：重要情感场景描写过于平淡

**修改方案**：
- 方案类型：增强
- 具体操作：增加情感描写的细节和深度
- 修改后内容：
  - 原文："她很伤心。"
  - 修改后："她心如刀绞，泪水夺眶而出，仿佛整个世界都在哭泣。"
- 操作说明：
  1. 增加生理反应描写（泪水、颤抖等）
  2. 增加心理活动
  3. 使用比喻、象征等修辞
  4. 环境烘托情感

### 情感积累不充分

**问题示例**：情感爆发前缺乏足够的积累

**修改方案**：
- 方案类型：增加
- 具体操作：增加情感积累的情节
- 修改后内容：
  - 在爆发前增加多个小的情感事件
  - 逐步升级情感强度
- 操作说明：
  1. 在爆发前增加2-3个情感事件
  2. 每个事件比前一个更强烈
  3. 形成情感阶梯

- [ ] 整体情感清晰
- [ ] 情感基调一致
- [ ] 情感转换自然
- [ ] 情感转换有铺垫
- [ ] 情感积累充分
- [ ] 情感爆发合理
- [ ] 能引起读者共鸣

---

## 严重程度分级

### A类（严重）
- 主要情感转换完全生硬，无铺垫
- 情感OOC（不符合角色性格）
- 重大情感场景描写失败

### B类（中等）
- 部分情感转换不够自然
- 情感表达不够充分
- 情感积累略显不足

### C类（轻微）
- 个别情感描写可以更细腻
- 个别情感转换可以更自然
